CatálogoApp

¿Cómo usar CatálogoApp?

  1. Opción A — Google Sheets: Abrí tu planilla → Archivo → Compartir → "Cualquiera con el link puede ver" → copiá el link y pegalo abajo.
  2. Opción B — CSV: Descargá la plantilla, completala y arrastrala a la zona de carga.
  3. Fotos: Si las fotos están en Google Drive, poné el link directo en la columna imagen_url. Si son locales, subí la carpeta.
  4. Generá el catálogo y usá Ctrl+P / Cmd+P para guardar como PDF.

💡 Si un producto no tiene SKU paquete, dejá esas celdas vacías — se muestra solo la caja. Las fotos de Drive deben ser compartidas como "cualquiera con el link".
